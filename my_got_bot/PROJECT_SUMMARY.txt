🚀 БЫСТРЫЙ СТАРТ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Установите зависимости:
   $ pip install requests python-dotenv

2. Создайте .env файл:
   $ cp .env.example .env

3. Добавьте ваш API ключ в .env:
   TOGETHER_API_KEY=your_real_key_here
   
   Получить ключ: https://api.together.xyz/settings/api-keys

4. Запустите бота:
   $ python main.py

   ИЛИ используйте автоматический скрипт:
   $ ./start.sh

🎯 ОСНОВНЫЕ ВОЗМОЖНОСТИ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ Итеративное размышление (Chain-of-Thought)
   → До 4 итераций на каждый запрос
   → Модель думает пошагово перед финальным ответом

✅ Краткосрочная память
   → Хранит последние 20 обменов
   → Бот помнит контекст разговора

✅ Модульная архитектура
   → Легко расширять и модифицировать
   → Каждый модуль = одна ответственность

✅ Гибкие промпты
   → Загружаются из .txt файлов
   → Легко редактировать без изменения кода

✅ Красивый вывод
   → Разделители для каждого модуля
   → Эмодзи-индикаторы состояния
   → Понятная визуализация процесса

✅ Обработка ошибок
   → Валидация входа
   → Корректная обработка API ошибок
   → Graceful shutdown (Ctrl+C)
